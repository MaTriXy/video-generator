# Simple Statement Scenes

## When to use
Read this file when the script has short punchy statements, bridge transitions between topics, emotional asides, whispered parentheticals, climactic one-liners, call-to-action endings, or outro scenes. These are minimal scenes where text IS the entire visual -- no complex diagrams, no UI mockups, just impactful text on screen.
