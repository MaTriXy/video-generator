# Metaphor & Physics Scenes

## When to use
Read this file when the script uses abstract concepts that need physical metaphors to visualize -- weight, gravity, orbits, spatial arrangements, forces, pressure, or when the narration describes something conceptual that can't be shown literally and needs a creative visual metaphor.
