# Screen Fill & Reveal Scenes

## When to use
Read this file when the script describes many identical or similar elements flooding the entire screen for dramatic effect, followed by a reveal, peel, sweep, or clearing effect. The pattern is: fill the canvas completely with repeated elements, then dramatically remove or transform them.
