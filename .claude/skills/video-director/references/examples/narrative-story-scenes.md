# Narrative & Story Scenes

## When to use
Read this file when the script tells a mini-story within a single scene -- a setup followed by an event and an outcome. These scenes have a narrative arc: something is established, something happens to it, and there is a resolution. The visual unfolds like a short story rather than displaying static information.
